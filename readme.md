## 🚀 Installing rentalx

To use, follow the steps:


- git clone https://github.com/rodrigorvsn/rentalx.git
- yarn || npm install
- Use `docker-compose-up` to create a image of server and database
- Use `yarn typeorm migration:run` to run migrations in database
